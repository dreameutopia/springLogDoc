# 环境配置

在开始使用 Spring 框架之前，您需要完成一些环境配置。本节将指导您设置开发环境。

## 安装 JDK

Spring 框架基于 Java，因此您需要安装 JDK（Java Development Kit）。推荐使用 JDK 17 或更高版本。

## 安装构建工具

您可以选择 Maven 或 Gradle 作为构建工具：

- **Maven**: 广泛使用的项目管理工具
- **Gradle**: 灵活的构建自动化工具

## 安装 IDE

推荐使用以下 IDE 之一：

- **IntelliJ IDEA**: 功能强大的 Java IDE
- **Eclipse**: 开源 IDE，有 Spring Tool Suite 插件
- **Visual Studio Code**: 轻量级编辑器，有 Spring 插件

## 验证环境

安装完成后，您可以通过以下命令验证环境：

```bash
java -version
mvn -version # 或 gradle -version
```
